# ecommerce_with_php

A new Flutter project.

## Using

- Dart/Flutter
- State Management (GetX)
- SQLite
- PHP & MySQL
- Firebase
- Google Map

## App Include

- Categories
- Items
- Cart
- Favorite
- Order tracking for each stage of live tracking (Google Map)
- Rating
- Coupon
- Notifications
- Payment methods with E wallet
- Web admin in order to control the application (Responsive)
